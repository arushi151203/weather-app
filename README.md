# 🌤️ Weather App

A personal project built during my internship at UptoSkills using HTML, CSS, and JavaScript. It fetches real-time weather data using the OpenWeatherMap API and displays it in a dark-themed, responsive UI with animated weather backgrounds.

## 📌 Features

- 🔍 Search weather by city name
- 📍 Detect current location and fetch local weather
- 🌡️ Displays temperature, feels like, min/max temp, and weather condition
- 💧 Shows humidity, wind speed & direction, pressure, and visibility
- 📅 Shows current date and day
- 🎨 Dynamic animated background changes based on weather condition (clear, rain, snow, thunderstorm, clouds, mist)
- 🌤️ Official weather icons from OpenWeatherMap

## 🛠️ Tech Stack

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/arushi151203/weather-app.git
   cd weather-app
   ```

2. Open `index.html` directly in your browser — no installation needed!

## 🔑 API Key

This project uses the OpenWeatherMap API. Replace the key in `script.js` with your own free key from:
- OpenWeatherMap — https://openweathermap.org/api

```javascript
const API_KEY = "YOUR_OPENWEATHERMAP_API_KEY";
```
